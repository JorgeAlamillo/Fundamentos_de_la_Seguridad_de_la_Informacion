### like1000

### Descripción

This [.tar file](https://challenge-files.picoctf.net/c_fickle_tempest/dc6349af33fec5413fe968b12506dde81953fdab2214acb3eaaf0b96cfff5dc4/1000.tar) got tarred a lot.
### Solución

Descargamos el archivo 1000.tar directamente a nuestra carpeta de trabajo en Kali Linux usando el comando wget para iniciar el análisis del reto.

Verificamos el tipo de archivo ejecutando el comando file 1000.tar y confirmamos que el sistema lo reconoce como un archivo comprimido tipo POSIX tar archive.

Inspeccionamos el contenido interno sin extraer nada todavía con el comando tar -tf 1000.tar para darnos cuenta de que dentro hay otro archivo llamado 999.tar, lo que indica un empaquetado múltiple de 1000 capas.

Creamos un ciclo automatizado en la terminal para no tener que descomprimir cada capa manualmente. Escribimos y ejecutamos el siguiente comando que realiza la extracción y borra el archivo anterior en cada paso: for i in {1000..1}; do tar -xf $i.tar && rm $i.tar ; done

Esperamos a que el proceso termine, lo cual nos dejará finalmente con un archivo llamado flag.png en el directorio después de haber eliminado las 1000 capas de archivos tar.

Confirmamos que el archivo final es una imagen válida usando nuevamente el comando file flag.png.

Abrimos la imagen resultante con el visor del sistema usando el comando eog flag.png o display flag.png para visualizar la bandera.
